# English Core 3000 · Part02

Rank 51–100（what … such），共 50 词。

## 51. what

- CEFR: A1
- POS: pron.; det.; adv.
- IPA (UK): /wɒt/
- Chinese: 什么；多么
- Collocation: what about;what if;what for
- Example: What is your favourite subject?
- 例句: 你最喜欢的科目是什么？
- Synonym: which;how
- MemoryTip: What a day! 感叹句很常见。

## 52. up

- CEFR: A1
- POS: adv.; prep.; adj.
- IPA (UK): /ʌp/
- Chinese: 向上；起来；完成
- Collocation: get up;wake up;give up
- Example: Please stand up when the teacher comes in.
- 例句: 老师进来时请起立。
- Synonym: upwards;aloft
- Antonym: down
- MemoryTip: 大量动词短语含 up，表完成或上升。

## 53. some

- CEFR: A1
- POS: det.; pron.; adv.
- IPA (UK): /sʌm/
- Chinese: 一些；某个；大约
- Collocation: some people;some of;for some time
- Example: Would you like some tea?
- 例句: 你想喝点茶吗？
- Synonym: a few;several
- Antonym: none;all
- MemoryTip: 肯定句多用 some，疑问/否定常改 any。

## 54. other

- Antonym: same
- MemoryTip: the other=两者中另一个；others=其他人/物。
