# English Core 3000 · Part06

Rank 251–300（idea … local），共 50 词。

## 251. idea

- CEFR: A1
- POS: noun
- IPA (UK): /aɪˈdɪə/
- Chinese: 想法；主意；概念
- Collocation: good idea;idea of;have an idea
- Example: That is a brilliant idea for the project.
- 例句: 那是这个项目的绝妙主意。
- Synonym: thought;notion;concept
- MemoryTip: have no idea = 完全不知道。
