# English Core 3000 · Part04

Rank 151–200（next … move），共 50 词。

## 151. next

- CEFR: A1
- POS: adj.; adv.; prep.; noun
- IPA (UK): /nekst/
- Chinese: 下一个；然后；旁边
- Collocation: next to;next week;the next
- Example: The next train leaves in five minutes.
- 例句: 下一班火车五分钟后发车。
- Synonym: following;then;adjacent
- Antonym: previous;last
- MemoryTip: next to = 紧挨着。

## 152. end

- CEFR: A1

- Synonym: finish;conclude;tip
- Antonym: start;beginning
- MemoryTip: in the end=最终；at the end of=在……末尾。
