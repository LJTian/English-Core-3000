# English Core 3000

原创英语核心词汇学习库（CEFR A1–B2），共 3000 词。

## 约定

- 词表：按 CEFR 分层自编（不复制 Oxford 等官方词表）
- 难度：约 A1 20% / A2 30% / B1 30% / B2 20%
- 拼写：仅英式
- 音标：仅 IPA_UK
- 每批 100 词，共 30 个 Part

## CSV 字段

```text
Rank,Word,CEFR,PartOfSpeech,IPA_UK,Chinese,Collocation,Example_EN,Example_CN,Synonym,Antonym,MemoryTip
```

## 目录

| 路径 | 说明 |
